# 🛡️ Parent Launcher

**Приложения под защитой — ярлыки в безопасности**

Программа для Windows, созданная специально для родителей, которые не хотят терять ярлыки любимых приложений. AppKeeper собирает все нужные программы в одном окне, защищая их от случайного удаления с рабочего стола.

## 🎯 Проблема, которую решает Parent Launcher

> «Сына, я не могу найти игру/приложение, я случайно удалил(а) с рабочего стола!»

Родители часто случайно удаляют ярлыки, а потом не могут найти приложения через поиск. AppKeeper сохраняет все важные программы в удобном окне, куда ребёнок всегда может зайти и запустить то, что нужно.

## ✨ Возможности

- 📌 **Сохранение ярлыков** — добавляйте любые приложения в один клик
- 🔍 **Быстрый поиск** — находите программу по имени за секунду
- 🖱️ **Удобное управление**:
  - Левая кнопка — запуск приложения
  - Правая кнопка — информация или удаление из списка
- 🗑️ **Безопасное удаление** — ярлык исчезает из программы, но приложение остаётся на компьютере
- 🔒 **Заблокированное перемещение** — иконки всегда на своих местах
- 🎨 **Чёрно-серый интерфейс** — строгий и понятный дизайн

## 📥 Установка

1. Скачайте `ParentLauncher_Setup.exe` из раздела Releases
2. Запустите установщик
3. Готово! Программа готова к работе

## 🖼️ Как это работает

1. Нажмите на кнопку **«+»** в правом верхнем углу
2. Выберите приложение из списка всех ярлыков на компьютере
3. Приложение появится в главном окне
4. Кликните на иконку — приложение запустится
