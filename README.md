# Родительский Лаунчер

![Python](https://img.shields.io/badge/Python-3.13-blue.svg)
![CustomTkinter](https://img.shields.io/badge/GUI-CustomTkinter-%231E90FF.svg)

**Простой и удобный лаунчер приложений специально для родителей и пожилых людей.**

Многие родители случайно удаляют ярлыки с рабочего стола и потом не могут найти нужные программы.  
Этот лаунчер решает проблему: все важные приложения собраны в одном красивом, понятном и защищённом окне.

Цель проекта — сделать использование компьютера максимально простым, безопасным и комфортным для родителей и пожилых пользователей.

### Основные возможности

- Удобная крупная сетка иконок
- Быстрый поиск по названию приложения
- Запуск программы одним кликом левой кнопкой мыши
- Правый клик по иконке → «Удалить» (только из лаунчера)
- Автоматическое сохранение списка приложений
- Современная тёмная тема
- Полностью переносимая версия + удобный установщик

---

### Технологии и библиотеки

- **Язык:** Python 3.13
- **Графический интерфейс - создание современного красивого GUI (тёмная тема, кнопки, поля ввода):** `customtkinter` (современная обёртка над Tkinter)
- **Работа с Windows ярлыками (.lnk):** `pywin32`
- **Обработка изображений и сохранение иконок в формате PNG:** `Pillow`
- **Извлечение иконок из исполняемых файлов (.exe, .dll):** `icoextract`

#### Установка зависимостей

Проект использует файл **`requirements.txt`** для удобной установки всех необходимых библиотек.
> [!IMPORTANT]
> Установку можно осуществить через cmd Windows или терминал VS Code или PowerShell
```powershell
# Установить все возможности разом
py -m pip install -r requirements.txt

# Ручная установка:
# Основные библиотеки
py -m pip install customtkinter pillow pywin32

# Для лучшего извлечения иконок (рекомендуется)
py -m pip install icoextract
```
Содержимое requirements.txt:
```
customtkinter
pillow
pywin32
icoextract
```

## Установщик

Чтобы программа была полностью переносимой и обычный пользователь (включая родителей) мог сам выбрать папку, куда установить программу, создать ярлыки в меню «Пуск» и на рабочий стол — необходим установщик.

Просто .exe файл, полученный через PyInstaller, не даёт такой возможности. Поэтому мы делаем два шага:
- [x] Собираем проект в один исполняемый файл (`ParentLauncher.exe`).
- [X] Создаём красивый графический установщик с помощью Inno Setup Complier.

#### Подробная инструкция по созданию установщика (актуально на апрель 2026):

*Шаг 1: Установка PyInstaller*
```Python
py -m pip install pyinstaller
```

*Шаг 2: Сборка исполняемого файла (.exe)*
В терминале VS Code необходимо выполнить след строку:
```Python
py -m PyInstaller --onefile --windowed --name "ParentLauncher" --icon assets/photo.ico --add-data "assets;assets" --collect-all app main.py
```
Пояснение каждой опции:

- **--onefile** - собирает всю программу в один единственный `.exe` файл (удобно для пользователя).
- **--windowed** - запускает программу без консоли (чтобы не было чёрного окна командной строки).
- **--name "ParentLauncher"** - задаёт имя итогового exe-файла.
- **--icon assets/photo.ico** - устанавливает иконку программы.
- **--add-data "assets;assets"** - добавляет папку assets внутрь exe (иконки, картинки и т.д.).
- **--collect-all app** - автоматически собирает все модули и файлы из папки app.
- **main.py** - главный файл, с которого начинается сборка.

После выполнения команды ВЫШЕ в корне проекта появится две папки:
* dist/ - здесь будет готовый `ParentLauncher.exe`
* build/ — временная папка (её можно удалить)

> [!IMPORTANT]
> Если у Вас возникли ошибки с каталогами или Вы внесли изменения в файлы сборки, после чего появляется необходимость удалить эти каталоги:
> ```Python
> Remove-Item -Recurse -Force build, dist
> ```

*Шаг 3: Установка Inno Setup*
1. Перейдите на официальный сайт: https://jrsoftware.org/isdl.php
2. Скачайте и установите Inno Setup (рекомендуется версия 6.3.x или новее).
3. После установки запустите программу Inno Setup Compiler.

> PS: Inno Setup — универсальный установщик для Windows. Следующий шаблон скрипта может работать работать с абсолютно любой средой и языком программирования. У вас должен быть один главный EXE-файл (или несколько), который Inno Setup скопирует на компьютер пользователя.

*Шаг 4: Создание скрипта установщика*

При открытии программы создаётет новый скрипт (New File → Create a new empty script file)
Затем вставляете следующий код:
```ini
[Setup]
DisableWelcomePage=no
DisableDirPage=no
WizardImageFile=C:\Users\User\Documents\ProgramList_PY\assets\ps1.bmp
AppName=Родительский Лаунчер
AppVersion=1.0
AppPublisher=Alex Skornyakov
AppPublisherURL=https://github.com/VortexCodeRU
DefaultDirName={autopf}\Родительский Лаунчер
DefaultGroupName=Родительский Лаунчер
OutputDir=Output
OutputBaseFilename=ParentLauncher_Setup
SetupIconFile=C:\Users\User\Documents\ProgramList_PY\assets\photo.ico
Compression=lzma
SolidCompression=yes
WizardStyle=modern

[Languages]
Name: "russian"; MessagesFile: "compiler:Languages\Russian.isl"

[Files]
Source: "C:\Users\User\Documents\ProgramList_PY\dist\ParentLauncher.exe"; DestDir: "{app}"; Flags: ignoreversion recursesubdirs createallsubdirs

[Icons]
Name: "{group}\Родительский Лаунчер"; Filename: "{app}\ParentLauncher.exe"
Name: "{commondesktop}\Родительский Лаунчер"; Filename: "{app}\ParentLauncher.exe"

[Run]
Filename: "{app}\ParentLauncher.exe"; Description: "Запустить Родительский Лаунчер"; Flags: nowait postinstall skipifsilent
```

Описание каждой секции:

- **[Setup]** - основные настройки программы (название, версия, иконка, путь установки).
- **[Languages]** - добавляет русский язык в установщик.
- **[Files]** - указывает, какой `.exe` файл нужно установить.
- **[Icons]** - создаёт ярлыки в меню «Пуск» и на рабочий стол.
- **[Run]** - предлагает запустить программу сразу после установки.

Сохраните скрипт как `ParentLauncher.iss` и нажмите кнопку Compile (F9) или синюю стрелочку "Старт".
Данный файл `.iss` можете сохранить куда угодно, НО *желательно* в папку с проектом. В каталоге OutPut появится готовый готовый установщик `ParentLauncher_Setup.exe`.

### Главный вопрос:
#### Почему эта программа лучше, чем просто папка с ярлыками на рабочем столе?

На первый взгляд может показаться, что достаточно создать папку «Мои программы» на рабочем столе и сложить туда ярлыки. Но на практике это решение имеет множество серьёзных недостатков, особенно когда компьютером пользуются родители или пожилые люди.
Вот почему наш лаунчер выигрывает по всем ключевым параметрам:
1. Защита от случайного удаления
   - Родители очень часто случайно удаляют ярлыки. Особенно при уборке рабочего стола или «наведении порядка».
   - После удаления они не знают, как найти программу заново — «где этот Интернет?», «куда делись Одноклассники?».
   - В лаунчере удаление ярлыка из программы не удаляет само приложение с компьютера. Можно спокойно нажать «Удалить», ничего не боясь.
2. Психологический комфорт
   - Пожилой человек видит не «кучу ярлыков на столе», а аккуратное приложение под названием «Родительский Лаунчер».
   - Это вызывает ощущение, что компьютер «разобрали и сделали удобным специально для него».
   - Значительно снижается тревога и страх «что-то сломать».
3. Всё в одном месте и ничего лишнего
   - На рабочем столе со временем скапливается огромное количество ярлыков, папок, документов и «всякого мусора».
   - В лаунчере пользователь видит только те программы, которые сам добавил. Ничего лишнего. Интерфейс остаётся чистым и понятным.
   - При случайном удалении ярлыка пользователю нечего бояться, так как при нажатии "Добавить" открывается путь в проводнике, в котором хранятся все ярлыки на компьютере Windows.
4. Удобный и понятный интерфейс для неподготовленного человека
   - Крупные иконки + названия под ними — легко читать даже людям с плохим зрением.
   - Есть строка поиска — достаточно начать набирать «Вайбер», «Одноклассники» или «Яндекс», и нужная программа сразу появляется.
   - Обычная папка на рабочем столе не имеет поиска и со временем превращается в свалку.
5. Дополнительные полезные функции
   - Правый клик по иконке → сразу можно удалить ярлык из лаунчера, если он больше не нужен.
   - Программа запоминает все добавленные приложения даже после перезагрузки компьютера.
   - Красивая тёмная тема, которая выглядит современно и не утомляет глаза.

Проще говоря, эта программа именно для того человека, который плохо разбирается в компьютере, разница огромная: от постоянного стресса и путаницы — к спокойствию и уверенности, что «всё нужное всегда под рукой»
